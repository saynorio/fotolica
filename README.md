# Fotolica

Fotolica is a photos app. It shows the users, their albums and photos with in it

## Tools used
- React Native
- Expo
- npm
- Visual Studio Code

### How to run the project
Make sure you have following things available on your system
- Node JS
- Expo
- npm
- Android Emulator or Real Android Device
